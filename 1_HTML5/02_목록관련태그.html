목록 관련 태그

ul(unordered list) 태그 : 순서가 없는 목록
ol(ordered list) 태그 : 순서가 있는 목록
li(list item) 태그 : ul, ol 태그의 자식으로 들어가 목록의 요소를 나타내는 태그

ol 태그

*ol 태그의 type 속성
-ol의 기본값은 숫자이다.
-type은 1, a, A, i, I를 지정할 수 있다.
(type = '1' 형식)

*ol태그의 start 속성
-시작 번호를 지정할 수 있다
(start='숫자' 형식)

*ol태그의 reversed 속성
-내림차순으로 적용 가능

  1. ordered
  2. list
  3. tag
